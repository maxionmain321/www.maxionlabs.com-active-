## Tech stack

GTM Systems "War Machine" Landing Page - High-performance Next.js marketing site with Linear/Vercel aesthetic.

### Framework & Runtime
- **Application Framework:** Next.js 15 (App Router)
- **Language/Runtime:** TypeScript, Node.js
- **Package Manager:** npm

### Frontend
- **JavaScript Framework:** React 19 (via Next.js 15)
- **CSS Framework:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Animation Library:** Framer Motion
- **Typography:** Geist Sans (or Inter fallback)

### Design System
- **Visual Style:** Linear.app / Vercel Aesthetic
- **Color Palette:** Pure black (#000000), slate borders (#1F1F1F), Indigo Glow (#5E5CE6)
- **Effects:** Glassmorphism, subtle pulsing glows, scroll-reveal animations
- **Layout:** Bento grids, 50/50 splits, interactive cards

### Key Components
- VSL Player with glassmorphism overlay
- Interactive 9-Cell Matrix with hover states
- Animated Bento Grid for deliverables
- Risk-reversal cards
- Testimonial marquee
- Shadcn Accordion for FAQ

### Deployment & Infrastructure
- **Hosting:** Vercel (recommended for Next.js)
- **CI/CD:** Vercel automatic deployments from Git

### Performance Requirements
- Fast page loads with Next.js optimizations
- Smooth Framer Motion animations
- Responsive across all devices
- SEO optimized for GTM/outbound marketing keywords
