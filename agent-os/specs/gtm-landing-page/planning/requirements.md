# **PRD: GTM Systems "War Machine" Landing Page**

## **1. Core Identity & Visual Language**

* **Visual Style:** Linear.app / Vercel Aesthetic. Pure black background (`#000000`), subtle slate borders (`#1F1F1F`), and "Indigo Glow" (`#5E5CE6`) for focus states.
* **Typography:** Geist Sans (or Inter) – High-tracking for headers, clean and clinical for body text.
* **Stack:** Next.js 15 (App Router), Tailwind CSS, Shadcn/UI, Framer Motion.
